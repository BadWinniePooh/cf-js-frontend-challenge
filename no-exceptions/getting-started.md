# Getting started — No Exceptions!

Run these steps once, before the [warmup](./README.md#warmup).

The code lives in [`./javascript/`](./javascript/), so all commands below run from that folder.

## 1. Install dependencies

```bash
cd javascript
yarn          # or: npm install
```

`yarn` is the default, but `npm` works just as well — pick whichever you prefer.

## 2. Run the tests

```bash
yarn test     # or: npm run test
```

All tests should be **green** before you start. If anything is red, fix the install before continuing — the exercise depends on starting from a known-good state.

## 3. Optional — lint

```bash
yarn lint
```

Lint should pass with no issues. Useful as a sanity check that your editor is picking up the project's eslint config.
